matplots.plot_profile(obs_seq, levels, type, bias=True, rmse=True, totalspread=True, depth=False)#
plot_profile on the levels for prior and posterior if present

  • bias

  • rmse

  • totalspread

For observations in pressure (Pa), the levels are assumed to be in hPa and the y-axis is in hPa and inverted. For ocean observations, which are height (m), set depth=True to invert y-axis.

Parameters:

  • obs_seq

  • levels

  • type

  • bias=True

  • rmse=True

  • totalspread=True

  • depth=False
