pyDARTdiags#

pyDARTdiags is a Python library for manipulating observation sequences and calculating observation-space diagnostics for the Data Assimilation Research Testbed (DART).

Caution

pyDARTdiags is under initial development, so please use caution. The MATLAB observation space diagnostics are available through DART.
