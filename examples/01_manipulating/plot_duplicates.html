Finding Duplicates in a Observation Sequence#

This example shows how to find duplicates in an observation sequence, and how to remove them.

Import the obs_sequence module

from pydartdiags.data import get_example_data

Read in the observation sequence file. In this example weâ€™ll use a real obs_seq file, the NCEP+ACARS.201303_6H.obs_seq2013030306 file. This is 6 hours of observations from March 3, 2013.

obs_seq = obsq.ObsSequence(data_file)
