Adding External Forward Operators to an Observation#

External, or ‚Äúprecomputed‚Äù forward operators are forward operators that are not calculated during filter, DART‚Äôs assimilation program. They are calculated externally, written to an observation sequence file, and read in by filter.

This example shows how to add an external forward operator to an observation. We‚Äôll be using observations for the Lorenz 96 tracer model, which is a toy model used to test assimilation algorithms. We are going to create some RAW_TRACER_CONCENTRATION forward operators.

Import the obs_sequence module, and numpy

import pydartdiags.obs_sequence.obs_sequence as obsq
from pydartdiags.data import get_example_data
import numpy as np

Read in the observation sequence file. In this example we‚Äôll use the obs_seq.out.tracer file. This file only has two observations.

data_file = get_example_data("obs_seq.out.tracer")
obs_seq = obsq.ObsSequence(data_file)

The external forward operator is stored in the ‚Äòexternal_FO‚Äô column. Currently, the ‚Äòexternal_FO‚Äô column is empty.

obs_seq.df.iloc[0]['external_FO']

[]

Let‚Äôs add a forward operator to the RAW_TRACER_CONCENTRATION observation. For this example, we‚Äôre going to use random numbers to create our ensemble of forward operators. For a scientific experiment, you would use appropriate forward operator results from your model. We want 80 ensemble members, so we‚Äôll create 80 random numbers, and we want tracer concentrations, so we‚Äôll force the numbers to be positive.

n = 80
random_numbers = np.abs(np.random.normal(loc=0.75, scale=0.1, size=n)).tolist()

DART observation sequence files have a specific format. DART expects the list of forward operators to have ‚Äòexternal_FO‚Äô followed by the number of ensemble members, followed by the ‚Äòkey‚Äô for the forward operator. The ‚Äòkey‚Äô is not used by DART, so we will just put 1 for the value.

random_numbers.insert(0, f'external_FO {n} 1')

Now we can add the forward operator to the ‚Äòexternal_FO‚Äô column for row 0

obs_seq.df.at[0,'external_FO'].extend(random_numbers)

Now the ‚Äòexternal_FO‚Äô column has the forward operator.
